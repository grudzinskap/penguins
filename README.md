## <span style="color:rgb(129,15,50)"><font face="verdana">**Instructions**</font></span>

*English version*

**Penguins!** is a simple puzzle-platformer game based on game named Lemmings.

* Getting started

To start the game, run ./penguins.py in a console. 

* How to play?

In the game there are three levels (easy, medium and hard), each limited to a single screen, with no scrolling. The objective of the game is to guide a group of penguins to the fridge (due to climat changes penguins need cold). Penguins walk forward bouncing off the walls. You can change each penguin behavior by clicking on one of three powerup buttons on the left side of the game window and then, clicking on the penguin. Here are powerup buttons function:

* umbrella - let a penguin fly, so it can fall safely even from a great height,
* stop - penguin stand still, other penguins that hit a stop-penguin simply reverse direction,
* bomb - bomb penguin die but a hole in snow is created (not working with the ground).

Your aim is to save at least 5/10 penguins from death. If you want to end the game, you can click on bomb button in down left corner of the screen, then all penguins die and game is over (it works only if all penguins are released).
